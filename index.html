--- title: Panacea ---

Panacea is a new way to generate Rails apps.

Panacea helps you to integrate the most common gems in your app so you can start the development proccess right away.

Get started!

First you need to install Panacea:

$ gem install panacea-rails

Then you are ready to start creating Rails apps via Panacea:

$ panacea your-app-name
